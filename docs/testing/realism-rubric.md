# Realism Rubric

Use browser review as a secondary sanity check after the trace score is computed.

## Categories

- `Off-ball shape`: players hold useful spacing and do not collapse into a constant scrum.
- `Ball interactions`: passes, receives, loose balls, and recoveries look readable on screen.
- `Defending`: pressure looks connected to the carrier instead of feeling random or absent.
- `Chance creation`: attacks visibly build toward danger instead of drifting without threat.

## Expectations

This is a `mini-figure football` rubric, not a full realism football rubric.

- Reward readable football-like motion.
- Tolerate abstraction and compressed timing.
- Penalize bunching, teleport-like transfers, dead attacks, and pressure that does not look connected to the ball.

## Browser Artifacts

Capture at minimum:

- one live-field screenshot
- one result screenshot
- one optional recording when testing a significant realism pass

Use these artifacts to confirm the trace score is not hiding visibly fake behavior.
